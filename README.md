# AT_tutor

獨立的後醫考古題自學網站。

## 已建置題庫

- 高雄醫學大學 115 學年度學士後醫學系招生考試
- 科目：物理及化學
- 題數：90 題（物理 45、化學 45）
- 原卷滿分：150 分

## 功能

- 原題高解析逐題截圖與點圖放大
- 五選一自動批改
- 隨手 15 題、物理分科、化學分科、正式模擬
- 作答後顯示官方答案與繁中詳解
- localStorage 練習歷程與錯題重練
- Firebase Firestore 交卷同步
- 教師登入、全班錯題／單元弱點、CSV 匯出

## 來源

使用者提供的 Google Drive 資料夾：

- `115高醫試題.pdf`
- `115高醫解答.pdf`

來源 PDF 中，高醫題本只包含物理及化學 90 題。答案 PDF 雖另含英文、生物及生化、計算機概論答案，但未提供相對應題本，因此網站不憑答案編造題目。

## 本機執行

```bash
python -m http.server 8765
```

開啟：

- 首頁：`http://127.0.0.1:8765/`
- 學生端：`http://127.0.0.1:8765/exams/kmu-115-physchem/`
- 教師端：`http://127.0.0.1:8765/exams/kmu-115-physchem/teacher.html`

## 資料與隱私

- 學生資料不寫入 Git repository。
- 公開題庫不包含真實學生資料。
- 教師讀取雲端資料需通過 Firebase Authentication 與 Firestore Rules。
- 原題版權屬原出題單位；本網站僅供授權範圍內教學與自學使用。
