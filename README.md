# AT_tutor

物理自學評量 GitHub Pages 平台。

## 已建置題庫

### 115學年度分科測驗物理｜高一高二範圍精選

- 來源：大學入學考試中心官方試卷與選擇題參考答案
- 篩選：高一必修物理＋高二選修物理I、II
- 題數／配分：13題、50分
- 題型：單選、多選、混合／非選
- 路徑：`exams/ceec-115-physics-g10-g11/`

### 高雄醫學大學115學年度學士後醫學系物理

- 純物理45題（原題號1–15、31–60）
- 物理滿分75分
- 路徑：`exams/kmu-115-physics/`

## 功能

- 高解析原題截圖與點圖放大
- 單選、多選與非選題作答
- 自動批改、官方多選部分給分
- 分年級練習、正式模擬、錯題重練
- 作答後顯示答案與繁中詳解
- localStorage練習歷程
- Firebase Firestore交卷同步與教師統計
- 可下載學生考卷與答案DOCX/PDF

## 本機執行

```bash
python -m http.server 8765
```

開啟：

- 首頁：`http://127.0.0.1:8765/`
- 115分科精選學生端：`http://127.0.0.1:8765/exams/ceec-115-physics-g10-g11/`
- 115分科精選教師端：`http://127.0.0.1:8765/exams/ceec-115-physics-g10-g11/teacher.html`

## 資料與版權

- `source/`與`work/`不發布。
- 公開題庫不包含真實學生資料。
- 各題庫使用獨立quizId；教師讀取雲端資料需通過Firebase Authentication與Firestore Rules。
- 原題版權屬原出題單位；本站標示官方來源，僅供教學與自學使用。
